# Stuff management app

This app is designed for selling stuff and managing orders.

![](doc/screenshots/main.png)
![](doc/screenshots/details.png)

## Installation

You need Node.js at least LTS version and MongoDB

```sh
npm install
```

## Usage

```sh
npm start
```
